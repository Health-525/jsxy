# 🐾 桌面宠物 - 使用说明

## 🚀 快速开始

### 方法一：使用启动脚本（推荐）
直接双击项目根目录下的 **`启动桌宠.bat`** 文件即可启动！

### 方法二：使用命令行
```bash
cd d:\新建文件夹\jsxy_pet\desktop-pet
npm start
```

## 📦 打包成可安装程序（可选）

如果你想把桌宠打包成一个 .exe 安装程序，这样更方便使用：

```bash
npm run build
```

打包后会在 `dist/` 目录下生成安装文件，安装后会自动在桌面创建图标！

## 🎮 功能说明

### 🐾 角色系统
- 默认使用 **3D 小狐狸**
- 右键点击桌宠 → 角色，可以切换不同角色
- 支持 3D 和 2D 两种模式！

### 💬 聊天助手
- 右键点击桌宠 → 聊天，可以和 AI 聊天
- 需要在设置中配置 AI 接口

### 🩺 健康提醒
- 定时喝水、久坐提醒
- 专注模式自动降低提醒频率
- 右键点击桌宠 → 立即测试健康提醒

### 🎯 实验性功能
- 体态和表情检测（需要打开摄像头权限）
- 在设置中可以开启和配置

## 🛠️ 开发说明

### 安装依赖
```bash
npm install
```

### 运行开发模式
```bash
npm run dev
```

### 代码检查
```bash
npm run check
```

## 💡 使用技巧

1. **拖拽移动**：点击并拖拽桌宠可以改变位置
2. **双击交互**：双击桌宠可以和它互动
3. **右键菜单**：所有功能都在右键菜单里！

## 📝 配置文件

所有配置会保存在项目目录下的 `.data/` 文件夹中
